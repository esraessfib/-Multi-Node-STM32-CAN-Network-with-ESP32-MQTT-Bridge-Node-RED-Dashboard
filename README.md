# -Multi-Node-STM32-CAN-Network-with-ESP32-MQTT-Bridge-Node-RED-Dashboard
Un système embarqué distribué où plusieurs STM32F4 communiquent via CAN bus, envoient leurs données à un nœud central qui les sauvegarde en SRAM Backup et les transfère vers un ESP32 MQTT bridge. Les données sont publiées sur Mosquitto et visualisées via Node-RED et via Bluetooth sur smartphone.
